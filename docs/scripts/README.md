# Скрипти розгортання та онбордингу

Усі шляхи відносні до **кореня репозиторію** (два рівні вгору від `docs/scripts/`).

| Файл | Опис |
|------|------|
| `dev.bat` / `dev.sh` | Встановлення залежностей: `npm install`. |
| `start-prod.bat` / `start-prod.sh` | Запуск додатка з `NODE_ENV=production`. |
| `run-prod.js` | Використовується командою `npm run start:prod`. |

Детальні інструкції: **`docs/deployment.md`**, для розробників — **`README.md`** у корені.
